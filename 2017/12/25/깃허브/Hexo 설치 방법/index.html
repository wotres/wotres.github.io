Hexo 설치 방법

앞서 깃에서 한 방법과 같이 깃허브에서 새로운 저장소로 username.github.io 저장소를 생성한다.
cmd 창에서 명령어를 입력해 Hexo를 설치한다.
아래 명령어를 통해 내가 원하는 경로에 내 블로그 정보를 다운로드한다.
git clone https://github.com/username/username.github.io
생성된 폴더에 접속후
echo “Hello World” > index.html // index.html에 Hello World 라는 단어 입력
git add –all
git commit -m “Initial commit”
git push -u origin master
위와 같은 순서대로 입력한 뒤
생성된 https://username.github.io 에 접속하여 확인한다.

이제 블로그를 관리한 blog 폴더를 만들어본다.
username.github.io 아래 blog 폴더를 생성한 뒤 아래 명령을 수행한다..
node.js를 설치하면 npm을 사용할 수 있는데, 패키지 중에 hexo-cli를 설치한다. (node.js 가 선행되어 설치되어야함)
npm install -g hexo-cli
설치 후
hexo -v
를 통해 버젼 확인
hexo init blog // blog은 아래 생성

_config.yml 파일을 열어 아래와 같은 식으로 입력한다.(note++과 같은 열리는 툴을 다운받은 뒤 활용한다)
title: Carl’s IT Tech Blog
subtitle: 매일 조금씩 나아가기 위한 기술 정리 블로그
description:
author: Carl
language: en
timezone: Asia/Seoul

그리고 아래 와 같은 부분에 내 블로그 정보를 디플로이 할 수 있게 입력한다.
url: http://wotres.github.io

deploy:
type: git
repo : https://github.com/wotres/wotres.github.io.git

##위와 같은 정보를 입력한 뒤 블로그에 파일을 반영한다.

  • 명령어
    hexo clean 을 통해 deploy 전에 정보를 정리한다.
    hexo generate 를 이용하면 github에 올라갈 리소스만 정리되어 따로 폴더가 생성된다.
    hexo deploy 를 통해 배포한다.
    보통은 편하게 아래와 같이 입력함
    hexo deploy –generate
    hexo d g 라고도 사용해도 됨…

시간이 조금 흐른 후에
생성된 https://username.github.io 에 접속하여 확인한다.
