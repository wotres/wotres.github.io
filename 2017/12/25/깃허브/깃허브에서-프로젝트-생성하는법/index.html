깃허브에서 프로젝트 생성하는법

  • https://github.com/ 접속
  • start a project 클릭후 저장소 생성 및 Repository name 에서 저장소 이름 설정
  • 명령순서
    깃에 사용자를 알려줌(파일이 누구의 것인지 정보를 달수있음)
    git config –global user.name “유저이름”
    git config –global user.email “유저 이메일”
    git init // 현재 폴더에 git 저장소를 생성함
    해당폴더에 파일 ex) python 파일 생성
    git status // 해당폴더 아래 파일이 반영되었는지 상태를 알려줌
    git add 생성파일.py // 해당 파일 추가
    git add . // 전체 add
    git commit -m “설명” // 깃에 반영 및 설명 부분에 변경사항에 대한 메시지 작성 가능
    // 반영됨
